# AIGC 提示词文档索引（开源协作版）

本目录用于存放本仓库的 AIGC 提示词与分析文档，面向开源协作场景。

## 目录规则

- 仅在 `aigc/prompts/`（含子目录）创建或更新提示词/文档。
- 不在仓库根目录创建提示词/文档。
- 文件名使用小写 kebab-case，中文文档使用 `.zh-CN.md` 后缀。

## 开源文档规范

- 使用可验证表述：结论应基于仓库当前代码与文档，不使用无法验证的断言。
- 明确上下文范围：注明分析依据（如“基于当前仓库快照”）。
- 保持中立语气：建议和结论应可讨论、可替换，不写命令式内部口径。
- 避免敏感信息：不写入凭据、私有地址、个人隐私与内部系统细节。
- 关注可复现性：给出可执行步骤、验证方式和边界条件。
